# **App Name**: MenuQR

## Core Features:

- Restaurant Login: Staff login to access restaurant management features.
- Menu Display: Display the restaurant's menu items with descriptions and prices.
- QR Code Scanner: Scan QR codes to identify restaurants.
- Manual Code Entry: Allow users to manually enter a restaurant code.
- Order Placement: Customers can place orders through the app by selecting menu items.
- Order Tracking: Restaurants can view and manage incoming orders in real time.

## Style Guidelines:

- Primary color: Vibrant coral (#FF7F50) to evoke warmth and appetite.
- Background color: Soft peach (#FFE5B4) as a light and inviting backdrop.
- Accent color: Sea green (#2E8B57) to provide contrast and a sense of freshness.
- Body and headline font: 'PT Sans' for a modern and readable interface.
- Use minimalist food-related icons.
- Clean, card-based layout for menu items.
- Subtle transitions on item selection and order confirmation.
